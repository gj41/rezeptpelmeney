Домашнии пельмени

Рецепт домашних пельменей:

Для теста:

  • Мука-300г

  • Кепяток-1 стакан(250г)

  • Соль-1ложка

    • Для начинки:

  • Свинина-200г

  • Говядина-200г

  • Лук репчатый-1 шт

  • Черный перец-1 щепотка

  • соль-1-2 ч.ложки

  • Молоко-0,5 стакана или вода-0,5 стакана

    • Пельмени со свино-говяжьим фаршем

    Для теста:

  • Вода горячая - 90-100 мл

  • Яйцо - 1 шт.

  • Масло растительное - 30 мл (2 ст. ложки)

  • Мука - 300 г + для работы с тестом

  • Соль - по вкусу

    • Для начинки:

  • Фарш свино-говяжий - 300 г

  • Лук репчатый - 70 г (1 шт.)

  • Чеснок - 1-2 зубчика

  • Соль - по вкусу

  • Перец чёрный молотый - по вкусу
